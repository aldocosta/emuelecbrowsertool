              EMULGA ATARI 7800 FULLSET - 65 GAMES

                              V2.0

 ****************************************************************
 *  VISIT  WWW.EMUL.GA  TO CHECK IF YOU HAVE THE LATEST VERSION *
 ****************************************************************

           All the roms come from the No-Intro romset.


 - - - - - -  G E N E R A L  I N F O R M A T I O N  - - - - - - -

This release is one of our custom drop-in romsets for Recalbox. After it has been downloaded, you just need to paste the files in the appropriate folder and refresh the gamelist. Everything is sorted, which means there will be no duplicate games and all games will be scraped. The version number for this set is written at the top so you can determine if a set has been updated that way you can come back later to see if you have the latest version, as we’ll be updating our romsets frequently.


 - - - - -   G A M E  S E L E C T I O N  P R O C E S S   - - - - -

Features USA romsets with English games in priority. Country exclusive releases from Europe, Australia, France, Germany, Spain, Italy, Netherlands, Brazil and Sweden included (in order). Country exclusive releases from Asia, Japan, China, Korea, Taiwan included and translated to English (in order). Japan systems contain English or playable games by an English speaker.

Roms are named with the no-intro naming convention. Roms are compressed with the highest level of compression. Multi cd-roms/diskettes games are in sub-folders to appear one time on your game list. Game name is as it is on the game box. Prototype and Unlicensed games are labeled in the game list. Clean gamelist.xml, doesn’t contain orphan entries.

Latest revision of a rom is prioritized. Unlicensed and prototype games are included only if they haven’t been released officially. Clone roms (demos, duplicates, alternate, public domain, pirate, hack, trained, etc) are excluded. Non-games such as action replay, test cartridge, program, etc are excluded.
